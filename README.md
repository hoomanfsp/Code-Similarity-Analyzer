# Code Similarity Checker

A web application to analyze code similarity and detect AI-generated content.

## Features
- Multi-dimensional similarity analysis
- AI generation detection
- Support for multiple programming languages
- Dark-themed modern UI

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt

2. Run the application:
bash
python app.py

3. Open browser to: http://localhost:5000

## Supported Languages
- Python (.py)
- Go (.go)
- Java (.java)
- C (.c)
- C++ (.cpp)
- C# (.cs)
- JavaScript (.js, .jsx)
- TypeScript (.ts, .tsx)

## Usage
1. Upload two code files with matching extensions
2. Click "Analyze Files"
3. View similarity scores and AI detection results
